simplepay is a Ruby gem which provides a programmatic interface to Amazon Simple Pay for dynamic form generation and IPN reception and content validation.

Released Version 0.1.0

Posted by Nathaniel Bibler

Version 0.1.0 has been released and is now available. The initial public release, marked as in beta, has been released. This provides the basic framework for form generation for both the Amazon Simple Pay Standard and yet-to-be-release Amazon Simple Pay Subscription services.

sudo gem install simplepay
